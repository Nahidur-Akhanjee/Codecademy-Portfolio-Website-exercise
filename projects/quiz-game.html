Case Study

Quiz Game

A browser-based quiz prototype that responds to answers, tracks progress, and gives users immediate feedback.

  • HTML
  • CSS
  • JavaScript
  • DOM Events

Project goal

The goal was to practice event handling and state updates in a small, approachable game interface that gives clear feedback after each answer.

Key features

  • Question and answer state management
  • Immediate correct or incorrect feedback
  • Progress indicator
  • Final score summary

What I practiced

This project reinforced DOM selection, click events, conditional logic, and using CSS states to make interaction feedback feel clear.
