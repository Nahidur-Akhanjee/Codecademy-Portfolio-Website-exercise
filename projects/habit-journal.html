Case Study

Habit Journal

A personal planning page designed to make weekly habit tracking feel simple, steady, and visually organized.

  • HTML
  • CSS
  • Responsive Layout
  • Accessibility

Project goal

The goal was to create a balanced content structure for daily habits, weekly notes, and progress reflections using only semantic HTML and CSS.

Key features

  • Responsive planner layout
  • Readable form-like sections
  • Consistent spacing and hierarchy
  • Accessible color contrast

What I practiced

This project helped me refine spacing, alignment, semantic section structure, and visual hierarchy for a focused everyday tool.
