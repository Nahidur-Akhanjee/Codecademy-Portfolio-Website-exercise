Case Study

Finance Tracker

A budgeting dashboard concept that helps users understand account totals, spending habits, and savings progress at a glance.

  • HTML
  • CSS
  • JavaScript
  • Responsive Design

Project goal

The goal was to design a clear dashboard for personal finance data without overwhelming the user. The layout prioritizes summaries, visual comparison, and fast scanning.

Key features

  • Account and savings summary cards
  • Category spending visuals
  • Responsive dashboard layout
  • JavaScript-driven filter states

What I practiced

This project strengthened my use of layout grids, reusable card styles, chart-like UI elements, and interface copy that keeps data easy to interpret.
